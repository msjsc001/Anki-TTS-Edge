# Anki-TTS-Edge 🎵➡️📚

基于Edge-TTS的Anki音频生成工具，快速为学习卡片添加高质量语音。

## ✨ 功能特性
- 一键生成Anki卡片音频
- 支持多种语音角色选择
- 简洁易用的图形界面
- 快速导入Anki媒体库

## 🛠️ 安装使用
1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python Anki-TTS-Edge.py
```

3. 界面操作：
- 输入要朗读的文本
- 选择语音类型
- 点击生成按钮
- 自动保存到Anki媒体文件夹
